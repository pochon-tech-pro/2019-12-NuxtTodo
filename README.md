# placticeNuxtTodo
NuxtでTodoアプリを作成

## 初期構築
```
$ npx create-nuxt-app .
```

## Build Setup
``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```
